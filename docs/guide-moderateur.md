# Guide du Modérateur — Xamle Civic

**Version 1.0 — Janvier 2025**

---

## Introduction

Ce guide est destiné aux modérateurs de la plateforme Xamle Civic. En tant que modérateur, vous êtes responsable de la qualité et de la fiabilité des informations publiées sur la plateforme.

---

## Vos responsabilités

1. **Valider les contributions citoyennes** soumises par les utilisateurs
2. **Modérer les commentaires** non appropriés
3. **Vérifier les sources** ajoutées aux politiques
4. **Signaler les anomalies** à l'équipe administrative

---

## Modération des contributions

### Workflow de validation

```
Soumission citoyenne → PENDING → [Modérateur] → APPROVED ou REJECTED
```

### Comment accéder aux contributions en attente

1. Connectez-vous sur [xamle.sn/auth/login](http://xamle.sn/auth/login)
2. Cliquez sur **Administration** > **Contributions**
3. Filtrez par statut **En attente**

### Critères d'approbation

| Critère | Description |
|---------|-------------|
| Pertinence | La contribution concerne bien la politique indiquée |
| Fiabilité | Le contenu est vérifiable ou témoignage crédible |
| Localisation | La région/commune mentionnée est cohérente |
| Conformité | Pas de discours haineux, d'insultes ou de désinformation |

### Processus de validation

**Pour approuver :**
1. Lire le contenu de la contribution
2. Vérifier la cohérence avec les données de la politique
3. Cliquer sur **Approuver**
4. Ajouter une note de modération (optionnel mais recommandé)

**Pour rejeter :**
1. Identifier le motif de rejet
2. Cliquer sur **Rejeter**
3. Obligatoire : écrire le motif dans le champ **Note de modération**
4. L'utilisateur recevra une notification avec votre note

### Motifs de rejet courants

- Contenu hors sujet
- Information non vérifiable
- Doublon d'une contribution existante
- Contenu inapproprié ou offensant
- Absence de localisation pour les témoignages terrain

---

## Modération des commentaires

### Commentaires à supprimer

- Insultes ou attaques personnelles
- Désinformation manifeste
- Spam ou publicité
- Contenu politique partisan non factuel

### Comment supprimer un commentaire

1. Naviguer vers la fiche politique concernée
2. Onglet **Commentaires**
3. Cliquer sur les 3 points (...) à côté du commentaire
4. Sélectionner **Supprimer**
5. Le commentaire est masqué (soft-delete, contenu archivé)

---

## Gestion des sources

### Niveaux de fiabilité des sources

| Niveau | Description |
|--------|-------------|
| **OFFICIAL** | Document officiel du gouvernement (PDF ministériel, DGPPE, etc.) |
| **VERIFIED** | Source vérifiée par l'équipe Xamle |
| **REPORTED** | Signalée par la communauté, en cours de vérification |
| **UNVERIFIED** | Ajoutée mais pas encore vérifiée |

### Archivage des liens

Pour chaque source ajoutée, pensez à cocher **Archiver le lien** afin de créer un snapshot de la page (protection contre les liens morts).

Toutes vos actions sont tracées dans le **Journal d'audit** (accessible uniquement aux administrateurs). Ce journal est immuable et ne peut pas être modifié.
